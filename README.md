## Features

🚀 Fast and responsive web interface
🎨 Dark/Light theme support
🔤 Mono/Sans font options
📱 Mobile-friendly design
🔍 Full-text search across all articles
🏷️ Save articles for later reading
👁️ Track read/unread status
🔐 Self-hosted for complete data control
📊 Clean, minimalist interface

## Tech Stack

Frontend: React
Backend: Node.js with Express
Database: SQLite
RSS Parser: rss-parser

Getting Started
Prerequisites

Node.js (v14 or higher)
npm or yarn

## Installation

Clone the repository:
`` git clone https://github.com/yourusername/clearfeed.git
cd clearfeed ``

`` # Install backend dependencies
cd server
npm install ``

`` # Install frontend dependencies
cd ../client
npm install ``

`` cd rss-feed
npm start ``

``cd frontend
npm start ``

The application will be available at http://localhost:3000
Usage

Add new feeds using the "Add Feed" button
Browse articles from all your feeds in one place
Filter articles by read/unread status
Save articles for later reading
Search across all your articles
Toggle between dark/light themes and mono/sans fonts
